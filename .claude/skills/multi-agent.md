启动多智能体自动协作系统。

**用法：**
- `/multi-agent` - 启用多智能体模式（本次对话有效）
- `/multi-agent <需求>` - 使用多智能体处理需求

**工作流程：**
1. 🧠 人性脑 - 识别意图与方向
2. 🔬 科学脑 - 逻辑与推理
3. 📚 学者脑 - 事实、规则、依据
4. ⚠️ 质疑官 - 风险与可行性
5. 🛠️ 执行者 - 整合与交付

**强制规则：**
- 必须按顺序执行全部5个智能体
- 不能跳过任何一个角色
- 不向用户询问或确认
- 自动补齐缺失信息
- 闭环完成思考

**示例：**
```
/multi-agent                # 启用多智能体模式
/multi-agent 创建一个组件   # 使用多智能体创建组件
```

**注意：**
启用后，所有需求都将自动通过5个智能体处理。
使用 `/multi-agent off` 可关闭模式。
